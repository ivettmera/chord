# Scripts de Automatización

Este directorio contiene scripts para demostración y experimentos automatizados.

## Archivos

### `demo.sh`
Script de demostración del sistema Chord DHT. Crea un ring local con múltiples nodos y ejecuta operaciones de ejemplo.

### `run-experiments.sh`
Script para ejecutar experimentos de escalabilidad y rendimiento. Incluye diferentes escenarios de prueba con métricas automáticas.

### Uso

```bash
# Ejecutar demostración básica
./scripts/automation/demo.sh

# Ejecutar experimentos de escalabilidad
./scripts/automation/run-experiments.sh
```

Estos scripts generan métricas en formato CSV y logs detallados para análisis posterior.
